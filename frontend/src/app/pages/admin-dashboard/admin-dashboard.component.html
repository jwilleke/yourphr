Administration tools. Available to admin users only; more cards will be added over time.

  Sandbox Testing

Connect to SMART-on-FHIR sandboxes (Epic, CMS Blue Button, …) with a bring-your-own client_id to test the connect → sync → display path. Admin-only — patients never see credentials.

  Provider Catalog
